# vslm-subnet-calculator
Calculate the number of subnets and hosts in a VSLM network.

This program is a VSLM (Variable-Length Subnet Mask) subnetting calculator simulator. 

It helps users to calculate the number of subnets and hosts in a VSLM, it was written in the context of the course "Computer Networks" at the National University of Costa Rica.
